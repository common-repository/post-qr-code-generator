= Display QR Code Auto Generator for every post & page below.

= How to use this plugin =

Install and activate the plugin. Now, in the WordPress post & page publish then after content showing the unique QR Code auto generate. 


= Theme compatibility =

All theme accepted this plugin

== Installation ==

1. Upload the entire post-qr-code-generator folder to the /wp-content/plugins/ directory or download the plugin via the Plugins section of the WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
